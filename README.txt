== Description ==
This plugin would enable you to directly sync your Orders with FlowPL for shipment fulfillment.

== Installation ==
● Make sure you have Woocommerce plugin installed and activated successfully. ● In order to upload the plugin, navigate to Plugins -> Add New. Then click the blue Upload Plugin button and provide the route to your extension \'shipsy-flowpl-econnect.zip\" archive and Click Install Now.  After successfully installing Activate the Plugin. ● On success, you should see a new item Shipsy Configuration in the menu items on the left side.
